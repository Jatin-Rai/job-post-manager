# Job Post Manager

## Getting Started

To get started with this project, follow these steps:

1. Clone this repository:

   ```bash
   git clone https://github.com/Jatin-Rai/job-post-manager.git

   ```

2. Install packages/dependencies:

   ```bash
   npm install
   or
   yarn
   ```

3. Run the application:

   ```bash
   npm run dev
   or
   yarn dev
   ```
